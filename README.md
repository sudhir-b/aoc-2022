# Advent of Code

Solutions for the Advent of Code puzzles, written in Rust.

## Structure

This project is a Cargo workspace. Each day's solution is in its own crate, e.g., `day1`, `day2`, etc.

## Running the solutions

To run the solution for a specific day, use the following command:

```bash
cargo run -p <day-name>
```

For example, to run the solution for day 1:

```bash
cargo run -p day1
```
